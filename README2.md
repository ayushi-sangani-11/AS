## Responsive Portfolio Webpage ##

This project is a fully responsive portfolio webpage designed using HTML, CSS, and JavaScript. The webpage automatically adapts to different screen sizes including mobile, tablet, and desktop devices.

Project Objective

The main objective of this project is to demonstrate seamless responsiveness across devices by creating a modern and user-friendly webpage layout.

Features
Responsive navigation bar
Hero section with introduction
Project showcase cards
Mobile hamburger menu using JavaScript
Responsive layouts using Flexbox and CSS Grid
Media queries for mobile and tablet optimization
Clean and modern UI design
Technologies Used
HTML5
CSS3
JavaScript
Responsive Design Techniques

This project uses:

Flexbox for layout alignment

Mobile-first responsive adjustments
