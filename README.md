# README – Nike Landing Page Redesign (Before → After)

## Project Overview

This project focuses on redesigning an existing Nike landing page to improve **usability, visual hierarchy, and overall aesthetics**. The original page contained multiple navigation items, promotional banners, and a content-heavy layout that created distractions. The redesigned version simplifies the experience and creates a more modern, product-focused interface.

---

## Design Goal

Create a cleaner and visually engaging landing page that:

* Improves user attention on the featured product
* Reduces visual clutter
* Enhances navigation clarity
* Creates a premium and modern look
* Improves overall user experience

---

# Before vs After Analysis

## Before (1nd Image – Existing Nike Website)

### Issues Identified:

* Too many navigation options at the top create visual overload.
* Multiple banners and promotional sections compete for attention.
* Large video/image area dominates the page without clear product focus.
* Lack of hierarchy makes users unsure where to look first.
* Search, navigation, sale banner, and hero section feel disconnected.

### User Experience Problems:

* Users may become distracted due to excessive content.
* Product visibility is weak.
* Important information competes with secondary elements.

---

## After (2st Image – Redesigned Landing Page)

### Improvements Made:

#### 1. Simplified Navigation

The navigation menu was reduced to only essential categories:

* Men
* Women
* Sale
* SNKRS

This minimizes cognitive load and makes navigation easier.

---

#### 2. Strong Visual Hierarchy

The redesigned page highlights:

* Product image
* Product title
* Product description
* Sale promotion

Large typography immediately draws user attention to the featured shoe.

---

#### 3. Product-Centered Design

The shoe becomes the primary focus through:

* Larger product placement
* Spacious layout
* Cleaner surrounding elements

This creates a stronger shopping experience.

---

#### 4. Modern Dark Aesthetic

The redesign introduces:

* Black-to-grey gradient background
* Minimal elements
* Floating product presentation

The dark theme creates a premium and stylish visual appearance.

---

#### 5. Better Promotional Placement

Instead of multiple banners, a single sale tag:

**“SALE UP TO 40%”**

is positioned near the hero section, making promotions visible without overwhelming the user.

---

## Design Decisions

| Design Element    |            Decision | Reason              |
| ----------------- | ------------------: | ------------------- |
| Navigation        |  Reduced menu items | Less clutter        |
| Background        |       Dark gradient | Premium feel        |
| Product Placement |  Center-right focus | Improve visibility  |
| Typography        | Large bold headings | Better hierarchy    |
| Promotions        |     Single sale tag | Clear communication |
| Layout            |   Minimal structure | Improved usability  |

---

## Tools Used

* Figma

## Conclusion

The redesigned Nike landing page transforms a content-heavy interface into a **minimal, product-driven experience**. By simplifying navigation, improving hierarchy, and enhancing aesthetics, the redesign creates a more focused and user-friendly shopping journey.
